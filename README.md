# J0han9's Neovim 配置

一个功能丰富的 Neovim 配置，专为现代开发体验而设计。

## 🚀 特性

### 核心功能
- **现代化界面**: 使用 Tokyo Night 主题，提供美观的深色主题
- **智能补全**: 集成 GitHub Copilot 和 blink.cmp 自动补全
- **LSP 支持**: 完整的语言服务器协议支持，提供代码智能提示
- **文件管理**: NvimTree 文件浏览器，支持 Git 集成
- **调试支持**: 集成 nvim-dap 调试器
- **AI 助手**: CopilotChat 提供 AI 编程助手功能

### 开发工具
- **代码格式化**: 自动代码格式化和语法检查
- **Git 集成**: LazyGit 提供 Git 操作界面
- **测试框架**: Neotest 支持多种测试框架
- **数据库工具**: nvim-dbee 数据库客户端
- **Markdown 预览**: 实时 Markdown 渲染

## 键位映射说明

### Colemak 布局
```
    ↑
    u
← n   i →
    e
    ↓
```

这个布局基于 Colemak 键盘布局设计，提供更符合人体工程学的移动方式。

### 模式说明
- **正常模式**: 默认的 Neovim 模式
- **插入模式**: 用于输入文本的模式
- **可视模式**: 用于选择文本的模式
- **全局**: 在所有模式下都有效

### Leader 键
- 默认 Leader 键设置为 `<Space>` (空格键)
- 所有以 `<Leader>` 开头的快捷键都需要先按空格键

## 🛠️ 安装与配置

### 前置要求
- Neovim 0.8.0 或更高版本
- Node.js 16.x 或更高版本 (用于 Copilot)
- Git

### 安装步骤
1. 克隆配置到 Neovim 配置目录：
```bash
git clone git@github.com:zywj0han9/nvim.git ~/.config/nvim
```

2. 启动 Neovim，插件将自动安装：
```bash
nvim
```

3. 等待 Lazy.nvim 完成插件安装

### LSP 服务器
配置会自动安装以下 LSP 服务器：
- pyright (Python)
- gopls (Go)
- clangd (C/C++)
- lua_ls (Lua)
- bashls (Bash)
- jsonls (JSON)
- vimls (Vim)
- typos_lsp (拼写检查)
- dockerls (Docker)
- sqls (SQL)
- cssls (CSS)
- html (HTML)
- markdown_oxide (Markdown)

## 📝 文件结构

```
~/.config/nvim/
├── init.lua                 # 主配置文件
├── lua/
│   ├── core/               # 核心配置
│   │   ├── option.lua      # Neovim 选项
│   │   └── keymap.lua      # 快捷键配置
│   ├── lib/                # 工具库
│   │   ├── icons.lua       # 图标定义
│   │   └── util.lua        # 工具函数
│   └── plugins/            # 插件配置
│       ├── lazy.lua        # 插件管理器
│       ├── list/           # 插件列表
│       └── config/         # 插件配置
└── README.md              # 说明文档
```

## 🔧 自定义配置

### 添加新插件
在 `lua/plugins/list/` 目录下创建新的插件配置文件。

### 修改快捷键
编辑 `lua/core/keymap.lua` 文件。

### 调整主题
修改 `lua/plugins/list/tokyonight.lua` 中的配置。
